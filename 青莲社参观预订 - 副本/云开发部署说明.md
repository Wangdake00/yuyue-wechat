# 微信云开发部署说明

## 一、准备工作

### 1. 开通云开发
1. 登录微信公众平台
2. 进入小程序管理后台
3. 点击"云开发"按钮开通云开发环境
4. 记录云开发环境ID

### 2. 配置云开发环境
在 `app.js` 中填写云开发环境ID：
```javascript
App({
  globalData: {
    env: "cloudbase-0gtahyfu3ab9e560"  // 在这里填写环境ID
  },
  // ...
});
```

## 二、数据库初始化

### 1. 创建集合
在微信开发者工具中：
1. 点击"云开发"按钮
2. 进入"数据库"标签
3. 创建以下集合：
   - `bookings` - 存储预约数据
   - `volunteers` - 存储志愿者信息

### 2. 导入志愿者数据
在控制台运行以下代码导入志愿者数据：

```javascript
const db = wx.cloud.database();

const volunteers = [
  { name: '张三', className: '药学2201班', phone: '13800138001' },
  { name: '李四', className: '药学2202班', phone: '13800138002' },
  { name: '王五', className: '中药学2201班', phone: '13800138003' },
  { name: '赵六', className: '中药学2202班', phone: '13800138004' },
  { name: '孙七', className: '制药工程2201班', phone: '13800138005' },
  { name: '周八', className: '制药工程2202班', phone: '13800138006' },
  { name: '吴九', className: '临床药学2201班', phone: '13800138007' },
  { name: '郑十', className: '临床药学2202班', phone: '13800138008' }
];

async function initVolunteers() {
  for (const v of volunteers) {
    await db.collection('volunteers').add({ data: v });
  }
  console.log('志愿者数据导入完成');
}

initVolunteers();
```

## 三、数据库权限设置

### 1. bookings 集合权限
设置为"仅创建者可读写"或"所有用户可读，仅创建者可写"

### 2. volunteers 集合权限
设置为"所有用户可读，仅管理员可写"

## 四、部署步骤

### 1. 上传代码
1. 在微信开发者工具中点击"上传"
2. 填写版本号和项目备注
3. 等待上传完成

### 2. 提交审核
1. 登录微信公众平台
2. 进入"版本管理"
3. 找到上传的版本，点击"提交审核"
4. 填写相关信息等待审核通过

### 3. 发布上线
审核通过后，点击"发布"即可上线

## 五、注意事项

1. **云开发免费额度**：注意查看免费额度使用情况，避免超额产生费用
2. **数据库索引**：如果数据量大，建议为常用查询字段创建索引
3. **数据备份**：定期导出重要数据备份
4. **权限控制**：生产环境请严格控制数据库权限

## 六、常见问题

### Q: 云开发环境ID在哪里找？
A: 在微信开发者工具中点击"云开发"按钮，在控制台页面可以看到环境ID。

### Q: 数据库查询失败怎么办？
A: 检查：
1. 云开发是否正确初始化
2. 集合名称是否正确
3. 数据库权限是否设置正确

### Q: 如何查看数据库中的数据？
A: 在微信开发者工具中点击"云开发"按钮，进入"数据库"标签即可查看和管理数据。
